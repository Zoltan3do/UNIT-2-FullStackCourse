ANIMAZIONI IN CSS

Le animazioni in CSS ( proprietà "animation") permettono un controllo molto più granulare dello scorrere dell'animazione rispetto ad una semplice transizione e consentono di collegare un'animazione ad un elemtno senza necessariamente dover sfruutare lo stato di hober; è possiblie infdatti collegare un'animazione ad un elemento all'avvio della pagina, programmaticamente alla pressione di un vottone , etc. queso è possiblie graize al fatto che cllegheremo l'animazione ad un aclasse css dedicata, e che quindi sarà possibile condizionare la sua aplicazione treamite javascript.
