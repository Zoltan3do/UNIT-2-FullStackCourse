CSS TRANSITIONS

Una transition è una "transizione" tra due valori di una proprietà CSS (o tra più proprietà CSS). La possiamo interpretare come un rallentamento nell'applicatione di una proprietà CSS. E' possibile "rallentare" anche l'applicazione di più proprietà CSS insieme. L'effetto di transizione solitamente viene associato all'evento di hover (al posizionamento del cursore sopra un elemento HTML).
